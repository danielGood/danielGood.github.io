Ray Cast Lighting Program

Ray Casting Algorithm

  1. A ray can be represented with a point and a vector. The point is the origin of the ray, and the vector represents the angle and maximum distance.
  2. Set the initial maximum distance for the ray, and the initial maximum distance point. 3.Iterate through all objects that need to be tested for intersection.
  3. Check for intersection between the ray and the object
    1. If there is an intersection between the ray and the object, get the distance to the origin
    2. If the new distance to the origin is smaller than the recorded one record it and the point of intersection.
  4. return intersection point
