Ray Casting Algorithm This algorithm returns the first intersection in a ray. It can be used for rendering and lighting. 3d raycast world. Ray tracers, a variant of ray casting, are used to render high quality objects. I will be using it in 2D to create lighting program.

  1. A ray can be represented with a point and a vector. The point is the origin of the ray, and the vector represents the angle and maximum distance.
  2. Set the initial maximum distance for the ray, and the initial maximum distance point.
  3. Iterate through all objects that need to be tested for intersection.
  4. Check for intersection between the ray and the object
    1. If there is an intersection between the ray and the object, get the distance to the origin
    2. If the new distance to the origin is smaller than the recorded one record it and the point of intersection.
  5. return intersection point

Variants: Use Octrees
